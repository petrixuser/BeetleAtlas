# Dokumentation

Dieser Ordner enthaelt die verbindliche Projektdokumentation fuer Beetle Box.

## Dateien

- `PFLICHTENHEFT.md`: Single Source of Truth fuer Anforderungen und Entscheidungen
- `ENTWICKLUNGSPLAN.md`: Modulplan, Phasen, Arbeitspakete und Abnahmekriterien
- `WORKLOG.md`: Laufender Arbeitsstand, offene Punkte und naechste Schritte

## Regel

Vor groesseren Aenderungen zuerst `PFLICHTENHEFT.md` und `WORKLOG.md` lesen.

Nach relevanten Aenderungen beide Dateien aktualisieren, falls Anforderungen, Entscheidungen
oder der Arbeitsstand betroffen sind.
